Иструкция по использованию программы "чат".

1) Для запуска сервера сервера войдите в каталог "chart". В каталоге "chart" наберите команду:
python Server -a <server name>
или
python Server
что эквивалентно: 
python Server -a localhost


2) Для запуска клиента сервера войдите в каталог "chart". В каталоге "chart" наберите команду:
python Client -a <server name>
или
python Client
что эквивалентно: 
python Client -a localhost

Система попросит ввести пароль доступа к серверу “Input server password:”. Внутри системы зашит 1 единственный стандартный пароль “Admin”. В рабочей версии можно сделать заложить много паролей и хранить их в файле или БД. Если пароль введен правильно, система возвращает уникальный код (открытый ключ GUID), который будет использоваться при дальнейшем взаимодействии с сервером вместо пароля. Система выдает пользователю следующее меню выбора: “message - key [1], exit - key [2]: 2”. Если пользователь нажимает клавишу [1], то дальше должен ввести свое сообщение для сервера. Выход и завершение работы с системой – клавиша [2].
Все сообщения клиентов записываются в текстовый файл Messages.txt, который лежит по следующему пути: chart\Server\Messages_from_Clients\Messages.txt
